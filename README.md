# Описание скрипта

Скрипт, приводящий в читабельный вид сплошную информацию находящуюся в JSON. 

# Использование

```python
from pprint_json import load_data

text = load_data(r"*****\4_json\als.json") #путь до файла json, который необходимо отформатировать
```
